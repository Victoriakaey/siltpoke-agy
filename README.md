# Siltpoke — Antigravity (agy) plugin

Install the review companion into [Antigravity (agy)](https://antigravity.google) in one line:

```bash
agy plugin install https://github.com/Victoriakaey/siltpoke-agy
```

This registers Siltpoke's Stop hook (`source: antigravity`) so agy's edits get reviewed. You still need a Siltpoke pet configured (`~/.siltpoke/config.json`) — see the main repo.
