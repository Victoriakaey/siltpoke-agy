# Siltpoke — Antigravity (agy) plugin

Install [Siltpoke](https://github.com/Victoriakaey/siltpoke) — a sassy ASCII buddy that reviews your coding agent's edits — into [Antigravity (agy)](https://antigravity.google) in one line:

```bash
agy plugin install https://github.com/Victoriakaey/siltpoke-agy
```

This registers Siltpoke's Stop hook (`source: antigravity`) so agy's edits get reviewed. You'll also need a Siltpoke pet configured (`~/.siltpoke/config.json`) — see the [main Siltpoke repo](https://github.com/Victoriakaey/siltpoke).

> This is the Antigravity distribution package for Siltpoke — a self-contained copy of Siltpoke's `agy` Stop hook + bundle. It is generated and synced on each Siltpoke release; don't edit it here.
